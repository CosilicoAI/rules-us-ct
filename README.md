# rules-us-ct

Connecticut rules in Akoma Ntoso XML format.

## Structure

- `statutes/` - Connecticut General Statutes
- `regulations/` - Regulations of Connecticut State Agencies (RCSA)

## Format

All documents are encoded in [Akoma Ntoso](http://www.akomantoso.org/) XML, an OASIS standard for legislative and legal documents.

## Source

- Statutes: [Connecticut General Assembly](https://www.cga.ct.gov/current/pub/titles.htm)
- Regulations: [eRegulations](https://eregulations.ct.gov/)
